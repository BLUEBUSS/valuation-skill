# 公司估值分析技能

## 一、估值方法选择矩阵

### 1.1 按行业属性选择

| 行业 | 首选方法 | 辅助方法 | 核心指标 | 特殊考量 |
|------|----------|----------|----------|----------|
| **科技/互联网** | EV/Revenue, EV/ARR | DCF, P/MAU | ARR, NRR, CAC, LTV | 用户价值、网络效应、研发资本化 |
| **金融-银行** | PB, DDM | P/PPOP | ROE, NIM, 不良率 | 监管资本要求、拨备覆盖率 |
| **金融-保险** | PEV, P/EV | PB | 内含价值、新业务价值 | 精算假设、投资收益率 |
| **金融-券商** | PB, PE | — | ROE, 经纪市占率 | 市场周期敏感性 |
| **制造业** | PE, EV/EBITDA | DCF, PB | 产能利用率、毛利率 | 周期位置、资本开支计划 |
| **消费-品牌** | PE, EV/EBITDA | DCF | 同店增长、品牌溢价 | 渠道结构、库存周转 |
| **消费-零售** | EV/EBITDA, PS | PE | 坪效、翻台率 | 租赁调整、加盟vs直营 |
| **医药-创新药** | rNPV, 管线估值 | PS | 管线阶段、成功概率 | 专利悬崖、研发投入 |
| **医药-仿制药** | PE, EV/EBITDA | DCF | 一致性评价进度 | 集采降价风险 |
| **能源/资源** | EV/EBITDA, PB | NAV, DCF | 储量、开采成本 | 大宗商品周期、ESG |
| **房地产** | NAV, PB | PE | 土储货值、去化率 | 融资成本、政策风险 |
| **公用事业** | DDM, PE | DCF | 股息率、ROE | 监管定价、特许经营权 |

### 1.2 按生命周期选择

| 阶段 | 判断标准 | 首选方法 | **禁用方法** | 关键假设 |
|------|----------|----------|-------------|----------|
| **初创期** | 无收入或收入<1亿，亏损，高速增长>100% | VC法、可比交易法、期权定价 | PE、PB、DCF | TAM渗透率、融资轮次 |
| **成长期** | 收入增长30-100%，亏损或微利，市占率提升 | DCF（远期盈利）、PS、EV/ARR | PE（若亏损） | 增长持续性、盈利拐点 |
| **成熟期** | 收入增长<20%，稳定盈利，高现金流 | PE、EV/EBITDA、DCF、DDM | **PS、EV/ARR** | 永续增长率、资本回报 |
| **衰退期** | 收入下滑，利润收窄，现金流下降 | PB、清算价值、资产重估 | PS、高PE | 资产变现、成本削减 |

> ⚠️ **重要**：成熟期公司已有稳定盈利，必须使用利润类指标（PE、EV/EBITDA），禁止使用收入类指标（PS、EV/ARR）

### 1.3 大型多元化公司：必须使用SOTP分部估值

**适用范围**：
- 大型互联网平台：腾讯、阿里、字节跳动、美团
- 科技巨头：苹果、微软、谷歌、亚马逊、Meta、英伟达、特斯拉
- 多元化集团：伯克希尔、复星、中信
- 判断标准：主营业务超过3个且各占收入20%以上

**SOTP方法**：
```
Step 1: 拆分业务板块（按收入/利润贡献）
Step 2: 为每个板块选择对应的纯业务可比公司
Step 3: 分别估值后加总
Step 4: 调整：+净现金 +投资组合NAV -控股折价(10-20%) -净债务
```

详见 `references/comparable_company_selection_and_multiple_adjustment.md`

### 1.4 快速决策流程

```
Step 1: 判断公司类型
        - 大型多元化公司 → 必须用SOTP分部估值
        - 单一业务公司 → 继续下一步

Step 2: 确定行业 → 查1.1表获取首选方法

Step 3: 判断生命周期 → 查1.2表确认/调整方法
        - 重要：检查禁用方法列，避免错误使用

Step 4: 检查特殊情况：
        - 亏损公司 → 不可用PE，改用PS/EV-Revenue/远期DCF
        - 成熟期公司 → 禁用PS，必须用PE/EV-EBITDA
        - 并购场景 → 必须用DCF评估协同效应
        - 高负债 → 优先EV类指标而非P类指标

Step 5: 选定主方法+辅助方法（至少2种交叉验证）
```

## 二、核心估值方法

### 2.1 DCF（现金流折现法）

**适用**：成长期/成熟期、现金流可预测的公司

**核心公式**：
```
企业价值 = Σ(FCFt / (1+WACC)^t) + TV / (1+WACC)^n

其中：
- FCF = EBIT×(1-税率) + 折旧摊销 - 资本支出 - 营运资金增加
- TV（终值）= FCFn×(1+g) / (WACC-g)  【永续增长法】
- TV（终值）= FCFn × 退出倍数        【退出倍数法】
```

**WACC计算**：详见 `references/wacc_calculation_and_adjustment.md`
```
WACC = (E/V)×Re + (D/V)×Rd×(1-T)
Re = Rf + β×ERP + 规模溢价 + 特定风险溢价
```

**参数区间参考**：
| 参数 | 保守 | 中性 | 乐观 | 说明 |
|------|------|------|------|------|
| 永续增长率g | 0-1% | 2-3% | 3-4% | 不应超过GDP长期增速 |
| WACC（成熟公司） | 10-12% | 8-10% | 6-8% | 按行业和风险调整 |
| WACC（成长公司） | 14-18% | 12-14% | 10-12% | 反映高不确定性 |
| 预测期 | 5年 | 7-10年 | 10年+ | 至稳定状态 |

### 2.2 相对估值法

**PE（市盈率）**
```
目标价 = EPS × 目标PE
目标PE = 可比公司PE中位数 × (1 + 增长率溢价) × (1 - 风险折价)
```
- 适用：盈利稳定的成熟公司
- 不适用：亏损公司、周期底部

**PB（市净率）**
```
目标价 = BVPS × 目标PB
合理PB = ROE / (COE - g)  【理论推导】
```
- 适用：金融、重资产、周期底部
- 核心：PB与ROE应匹配，高ROE支撑高PB

**PS（市销率）**
```
目标市值 = 营收 × 目标PS
目标PS = 可比公司PS × (目标毛利率/可比毛利率) × 增长率调整
```
- 适用：**仅限成长期**亏损但有收入的公司
- **禁用**：成熟期盈利公司（必须用PE/EV-EBITDA）
- 注意：必须考虑毛利率差异

> ⚠️ **警告**：PS法仅适用于成长期亏损公司。对成熟期盈利公司使用PS会导致严重高估。

**EV/EBITDA**
```
目标EV = EBITDA × 目标倍数
股权价值 = EV - 净债务 - 少数股东权益 + 联营投资价值
```
- 适用：资本密集、高折旧、跨国比较
- 优势：消除资本结构和折旧政策差异

**可比公司选择**：详见 `references/comparable_company_selection_and_multiple_adjustment.md`

### 2.3 DDM（股利折现模型）

**适用**：稳定分红的成熟公司（银行、公用事业、REITs）

**Gordon模型**：
```
股票价值 = D1 / (r - g)
其中：D1 = 下期股息，r = 股权成本，g = 股息增长率
```

**两阶段DDM**：
```
P = Σ(Dt/(1+r)^t) + Pn/(1+r)^n
Pn = Dn+1/(r-g)  【第二阶段进入稳定增长】
```

### 2.4 行业特殊方法

**NAV（净资产价值法）** - 房地产
```
NAV = Σ(项目货值 - 开发成本 - 税费) / (1+r)^t + 净现金 - 净债务
每股NAV = NAV / 总股本
```

**PEV（内含价值法）** - 保险
```
PEV = EV + 新业务价值×倍数
EV = 调整后净资产 + 有效业务价值
```

**rNPV（风险调整净现值）** - 创新药
```
rNPV = Σ(成功概率 × 峰值销售 × 净利率 × 折现因子) - 研发投入现值
```

## 三、行业估值要点速查

### 3.1 科技/互联网

**估值要点**：
- 用户指标：MAU、DAU、ARPU、留存率
- SaaS指标：ARR、NRR(>120%优秀)、CAC回收期(<12月健康)、LTV/CAC(>3优秀)
- 变现效率：货币化率、广告加载率、付费率

**方法选择**：
- 未盈利：EV/ARR（8-15x）、EV/Revenue（3-8x）、P/MAU
- 已盈利：PE、EV/EBITDA、DCF

**参考倍数**（A股/港股）：
| 类型 | EV/Revenue | EV/ARR | PE（盈利后） |
|------|------------|--------|-------------|
| SaaS | 5-15x | 8-20x | 30-50x |
| 电商平台 | 1-3x | — | 20-35x |
| 游戏 | 2-5x | — | 15-25x |

详见 `references/saas_and_tech_valuation_deep_dive.md`

### 3.2 金融-银行

**估值要点**：
- 资产质量：不良率、关注类占比、拨备覆盖率
- 盈利能力：NIM、ROE、成本收入比
- 资本充足：核心一级资本充足率

**方法选择**：PB为主，DDM辅助

**PB-ROE框架**：
| ROE | 合理PB |
|-----|--------|
| <8% | 0.4-0.6x |
| 8-12% | 0.6-0.9x |
| 12-15% | 0.9-1.2x |
| >15% | 1.2-1.5x |

### 3.3 金融-保险

**估值要点**：
- 内含价值EV：调整后净资产 + 有效业务价值
- 新业务价值NBV：新保单创造的价值
- 新业务价值率：NBV/新单保费

**方法选择**：P/EV为主

**参考倍数**：
| 类型 | P/EV |
|------|------|
| 寿险（优质） | 0.8-1.2x |
| 寿险（一般） | 0.5-0.8x |
| 财险 | 0.8-1.5x（看综合成本率） |

### 3.4 制造业

**估值要点**：
- 周期位置判断：产能利用率、库存水平、产品价格
- 成本结构：原材料占比、人工成本、折旧
- 竞争格局：市占率、产能扩张计划

**方法选择**：
- 周期顶部：PB（避免PE陷阱）
- 周期底部：PB、正常化PE
- 稳态：PE、EV/EBITDA

**周期调整**：使用3-5年平均盈利计算正常化PE

### 3.5 消费

**估值要点**：
- 品牌消费：品牌力、提价能力、渠道结构
- 零售：同店增长、坪效、租金占比、库存周转

**方法选择**：PE、EV/EBITDA

**参考倍数**（A股）：
| 类型 | PE | EV/EBITDA |
|------|-----|-----------|
| 高端白酒 | 25-40x | 20-30x |
| 大众消费 | 15-25x | 10-15x |
| 零售 | 15-25x | 8-12x |

### 3.6 医药

**估值要点**：
- 创新药：管线价值、临床阶段、适应症空间
- 仿制药：一致性评价进度、集采影响

**方法选择**：
- 创新药：rNPV、管线分部估值
- 仿制药：PE、EV/EBITDA

**管线成功概率参考**：
| 阶段 | 成功概率 |
|------|----------|
| 临床前 | 5-10% |
| Phase I | 15-20% |
| Phase II | 25-35% |
| Phase III | 50-70% |
| 已上市 | 90%+ |

### 3.7 能源/资源

**估值要点**：
- 储量：证实储量、可能储量
- 成本：完全成本、现金成本
- 周期：大宗商品价格位置

**方法选择**：EV/EBITDA、NAV、PB

**周期判断**：当前价格 vs 长期均价 → 决定用当期还是正常化盈利

### 3.8 房地产

**估值要点**：
- 土储：货值、分布、获取成本
- 销售：去化率、回款率
- 财务：净负债率、融资成本

**方法选择**：NAV为主，PB辅助

**NAV折价参考**：
| 公司类型 | NAV折价 |
|----------|---------|
| 龙头央企 | 0-20% |
| 优质民企 | 20-40% |
| 高杠杆民企 | 40-60% |

### 3.9 公用事业

**估值要点**：
- 特许经营权：期限、区域
- 监管定价：调价机制
- 分红：股息率、分红比例

**方法选择**：DDM、PE

**参考指标**：
- 股息率：4-6%为合理
- PE：10-15x

## 四、估值输出规范（券商研报格式）

### 4.1 估值报告结构

```
一、估值摘要
   - 目标公司：[名称]（[代码]）
   - 估值日期：[YYYY-MM-DD]
   - 当前股价：[价格]
   - 目标价格：[价格]（[上涨/下跌空间]%）
   - 投资评级：[买入/增持/持有/减持/卖出]
   - 估值方法：[主方法] + [辅助方法]

二、关键假设
   | 假设项 | 数值 | 依据 |
   |--------|------|------|
   | ... | ... | ... |

三、估值计算
   3.1 [主方法]估值
       - 计算过程
       - 结果：[X]元
   3.2 [辅助方法]估值
       - 计算过程
       - 结果：[Y]元
   3.3 综合估值
       - 权重分配：主方法[X]%，辅助方法[Y]%
       - 综合目标价：[Z]元

四、敏感性分析
   | 核心变量 | -20% | -10% | 基准 | +10% | +20% |
   |----------|------|------|------|------|------|
   | 目标价 | ... | ... | ... | ... | ... |

五、估值对比
   | 可比公司 | 当前PE | 当前PB | 目标公司 |
   |----------|--------|--------|----------|
   | ... | ... | ... | ... |

六、风险提示
   - [风险1]：[影响说明]
   - [风险2]：[影响说明]
```

### 4.2 投资评级标准

| 评级 | 定义 |
|------|------|
| **买入** | 未来6-12个月预期涨幅 > 15% |
| **增持** | 预期涨幅 5-15% |
| **持有** | 预期涨幅 -5% 至 +5% |
| **减持** | 预期跌幅 5-15% |
| **卖出** | 预期跌幅 > 15% |

### 4.3 关键输出要素检查清单

- [ ] 目标价和投资评级
- [ ] 主要估值方法及计算过程
- [ ] 关键假设及依据
- [ ] 可比公司对比
- [ ] 敏感性分析（至少2个核心变量）
- [ ] 主要风险提示（至少3条）

## 五、估值校验与风险控制

### 5.1 交叉验证

- 使用至少2种不同逻辑的方法
- 结果偏差应<20%，否则需检查假设
- DCF与相对估值互相验证

### 5.2 结果合理性检验（重要）

**估值区间检验**：

| 检查项 | 合理标准 | 异常处理 |
|--------|----------|----------|
| 各方法估值偏差 | **<±30%** | 偏差>30%需剔除异常方法或检查假设 |
| 最终估值区间 | 上下限差距<50% | 区间过大说明方法不一致，需重新审视 |
| vs当前股价 | 偏差<±50% | 偏差过大需特别说明原因 |

**隐含指标检验**：

| 检查项 | 合理标准 |
|--------|----------|
| 永续增长率 | 不超过GDP+通胀（2-4%） |
| 隐含PE | 在行业历史10%-90%分位内 |
| 隐含增长率 | 不超过行业天花板 |
| WACC | 与同行业公司差异<2% |

**异常值处理**：
- 若某方法估值超过其他方法2倍以上 → **必须剔除**或单独说明
- 若PS估值远高于PE估值 → 检查是否错误使用PS（成熟期禁用）
- 最终估值应取**中位数或加权平均**，而非简单平均

### 5.3 币种统一规则

| 上市地 | 计价货币 | 说明 |
|--------|----------|------|
| A股 | **人民币（CNY）** | 固定 |
| 港股 | **港币（HKD）** | 固定 |
| 美股 | **美元（USD）** | 固定 |
| 港美股双上市 | **美元（USD）** | 如阿里巴巴，统一用美元 |
| A+H股双上市 | **人民币（CNY）** | 以A股为主 |

**汇率处理**：
- 必须在报告中明确标注使用的货币单位
- 跨市场可比公司需统一折算，并注明汇率
- 汇率取估值日的即期汇率

### 5.4 常见错误规避

- **锚定偏差**：不要从股价反推假设
- **幸存者偏差**：可比公司需包含表现差的
- **周期陷阱**：周期顶部不要用当期PE
- **增长陷阱**：高增长不能无限外推
