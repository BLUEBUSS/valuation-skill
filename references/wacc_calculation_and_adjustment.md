# WACC计算与动态调整详解

> WACC（加权平均资本成本）是DCF估值的核心参数，直接影响估值结果。
> 本文档详细阐述WACC的计算方法、参数获取和常见调整。

---

## 一、WACC核心公式

### 1.1 基本公式

```
WACC = (E/V) × Re + (D/V) × Rd × (1 - T)

其中：
- E = 股权市值
- D = 债务市值
- V = E + D（企业总价值）
- Re = 股权成本
- Rd = 债务成本
- T = 企业所得税率
```

### 1.2 股权成本（Re）计算 - CAPM模型

```
Re = Rf + β × ERP + SP + α

其中：
- Rf = 无风险利率
- β = 股权Beta系数
- ERP = 股权风险溢价（Equity Risk Premium）
- SP = 规模溢价（Size Premium，小公司额外风险）
- α = 公司特定风险溢价（可选）
```

## 二、各参数详解与获取方法

### 2.1 无风险利率（Rf）

**定义**：理论上无违约风险的收益率

**获取方法**：

| 市场 | 推荐基准 | 当前参考 | 数据来源 |
|------|----------|----------|----------|
| A股 | 10年期国债收益率 | 2.5-3.0% | 中国债券信息网、Wind |
| 港股 | 10年期美债或港债 | 4.0-4.5% | Bloomberg、Wind |
| 美股 | 10年期美债收益率 | 4.0-4.5% | FRED、Bloomberg |

**注意事项**：
- 使用估值日当天或近期平均值
- 预测期较长时（>10年），可考虑使用30年国债收益率
- 跨境估值需匹配现金流币种

### 2.2 股权风险溢价（ERP）

**定义**：投资股票相对于无风险资产所要求的额外回报

**获取方法**：

| 方法 | 说明 | A股参考 |
|------|------|---------|
| 历史ERP | 股市历史回报 - 国债回报 | 5-7% |
| 隐含ERP | 从当前市场PE反推 | 4-6% |
| 学术研究 | Damodaran等数据库 | 5-6% |
| **建议使用** | 综合判断 | **6%** |

**Damodaran数据库参考**（2024年）：
- 中国：6.5%
- 美国：4.6%
- 新兴市场平均：7-8%

### 2.3 Beta系数（β）

#### 2.3.1 Beta的含义

```
β = Cov(Ri, Rm) / Var(Rm)

- β > 1：股票波动大于市场
- β = 1：与市场同步
- β < 1：股票波动小于市场
```

#### 2.3.2 获取原始Beta

**方法一：直接获取**
- Wind、Bloomberg等终端直接查询
- 通常使用周收益率、2-5年数据

**方法二：回归计算**
```
Ri = α + β × Rm + ε

其中：
- Ri = 个股收益率
- Rm = 市场指数收益率（如沪深300）
```

#### 2.3.3 去杠杆（Unlevered Beta）

**目的**：剔除财务杠杆影响，得到纯业务风险

**Hamada公式**：
```
βu = βL / [1 + (1-T) × (D/E)]

其中：
- βu = 无杠杆Beta（资产Beta）
- βL = 有杠杆Beta（股权Beta）
- T = 税率
- D/E = 债务/股权比率
```

**示例**：
```
某公司：βL = 1.5，D/E = 0.5，T = 25%
βu = 1.5 / [1 + (1-25%) × 0.5]
   = 1.5 / 1.375
   = 1.09
```

#### 2.3.4 再杠杆（Relevered Beta）

**目的**：根据目标公司资本结构重新加杠杆

**公式**：
```
βL = βu × [1 + (1-T) × (D/E)]
```

**示例**：
```
使用行业平均βu = 1.1，目标公司D/E = 0.8，T = 25%
βL = 1.1 × [1 + (1-25%) × 0.8]
   = 1.1 × 1.6
   = 1.76
```

#### 2.3.5 行业Beta参考

| 行业 | 资产Beta（βu） | 参考股权Beta（βL） |
|------|---------------|-------------------|
| 公用事业 | 0.4-0.6 | 0.5-0.8 |
| 银行 | 0.7-0.9 | 1.0-1.3 |
| 消费必需 | 0.6-0.8 | 0.7-1.0 |
| 制造业 | 0.8-1.0 | 1.0-1.3 |
| 医药 | 0.9-1.1 | 1.0-1.4 |
| 科技 | 1.1-1.4 | 1.3-1.8 |
| 券商 | 1.2-1.5 | 1.5-2.0 |

> 详细行业Beta参考见 `references/industry_parameters.md`

### 2.4 规模溢价（Size Premium）

**依据**：小公司风险更高，需额外补偿

| 市值规模 | 规模溢价 |
|----------|----------|
| >1000亿 | 0% |
| 200-1000亿 | +0.5-1.0% |
| 50-200亿 | +1.0-2.0% |
| <50亿 | +2.0-4.0% |

### 2.5 债务成本（Rd）

**获取方法**：

| 情景 | 方法 |
|------|------|
| 有公开债券 | 使用债券到期收益率（YTM） |
| 有银行贷款 | 使用最新贷款利率 |
| 无公开数据 | 参考同评级公司债利率 |

**信用评级与债务成本参考**（中国市场）：

| 评级 | 信用利差 | Rd参考（Rf=3%） |
|------|----------|----------------|
| AAA | 0.5-1.0% | 3.5-4.0% |
| AA+ | 1.0-1.5% | 4.0-4.5% |
| AA | 1.5-2.0% | 4.5-5.0% |
| AA- | 2.0-3.0% | 5.0-6.0% |
| A+ | 3.0-4.0% | 6.0-7.0% |

### 2.6 资本结构（D/E）

**使用原则**：
- 优先使用**目标资本结构**
- 其次使用**行业平均资本结构**
- 最后使用**当前资本结构**

**注意事项**：
- 股权E使用市值，非账面值
- 债务D使用账面值（若无市值数据）
- 考虑租赁负债资本化（IFRS 16）
- 考虑优先股、可转债等混合工具

## 三、WACC计算完整示例

### 3.1 案例：某制造业公司

**基础数据**：
- 股票市值：200亿元
- 有息负债：50亿元
- 税率：25%
- 公司评级：AA
- 市值规模：中盘

**Step 1：计算股权成本（Re）**
```
Rf = 2.8%（10年国债）
βL = 1.2（从Wind获取）
ERP = 6%
SP = 1%（中盘规模溢价）

Re = 2.8% + 1.2 × 6% + 1%
   = 2.8% + 7.2% + 1%
   = 11.0%
```

**Step 2：计算债务成本（Rd）**
```
AA评级对应利差 = 1.5%
Rd = 2.8% + 1.5% = 4.3%
税后Rd = 4.3% × (1-25%) = 3.2%
```

**Step 3：计算权重**
```
E = 200亿
D = 50亿
V = 250亿

E/V = 200/250 = 80%
D/V = 50/250 = 20%
```

**Step 4：计算WACC**
```
WACC = 80% × 11.0% + 20% × 4.3% × (1-25%)
     = 8.8% + 0.6%
     = 9.4%
```

### 3.2 案例：高成长科技公司

**基础数据**：
- 股票市值：50亿元
- 有息负债：5亿元（几乎无负债）
- 税率：15%（高新技术企业）
- 小盘股

**Step 1：计算股权成本（Re）**
```
Rf = 2.8%
行业平均βu = 1.2
公司D/E = 5/50 = 0.1
βL = 1.2 × [1 + (1-15%) × 0.1] = 1.30

ERP = 6%
SP = 2.5%（小盘溢价）
α = 2%（公司特定风险：依赖大客户）

Re = 2.8% + 1.30 × 6% + 2.5% + 2%
   = 2.8% + 7.8% + 4.5%
   = 15.1%
```

**Step 2：计算WACC**
```
E/V = 50/55 = 91%
D/V = 5/55 = 9%
Rd = 5%

WACC = 91% × 15.1% + 9% × 5% × (1-15%)
     = 13.7% + 0.4%
     = 14.1%
```

## 四、特殊情况调整

### 4.1 负债为负（净现金）

```
当净现金 > 0时：
- 方案一：WACC = Re（假设全股权融资）
- 方案二：使用行业平均资本结构计算WACC
```

### 4.2 跨国公司

```
多币种现金流：
- 分别用各币种的Rf和WACC
- 或统一折算为一种货币

新兴市场子公司：
- 加入国家风险溢价（Country Risk Premium）
- 参考CDS利差或主权评级利差
```

### 4.3 周期性行业

```
周期位置调整：
- 周期底部：使用较高的β（历史高位）
- 周期顶部：使用正常化β
```

### 4.4 高速增长公司

```
分阶段WACC：
- 高增长期：使用较高WACC（反映高风险）
- 稳定期：切换到较低WACC（接近成熟公司）
```

## 五、敏感性分析

### 5.1 WACC敏感性测试

**标准模板**：

| WACC变动 | -2% | -1% | 基准 | +1% | +2% |
|----------|-----|-----|------|-----|-----|
| 企业价值变动 | +25% | +12% | 0% | -10% | -18% |

> 注：以上为示例，实际敏感性取决于增长率和预测期长度

### 5.2 WACC vs 永续增长率交叉敏感性

| g \ WACC | 8% | 9% | 10% | 11% | 12% |
|----------|-----|-----|------|-----|-----|
| 1% | 128 | 113 | 100 | 89 | 80 |
| 2% | 150 | 129 | 111 | 97 | 85 |
| 3% | 180 | 150 | 125 | 106 | 91 |
| 4% | 225 | 180 | 143 | 118 | 100 |

> 单位：相对基准（WACC=10%, g=2%）的比例

## 六、WACC计算检查清单

- [ ] 无风险利率是否与现金流币种匹配
- [ ] Beta是否使用了去杠杆/再杠杆调整
- [ ] 是否考虑了规模溢价
- [ ] 债务成本是否反映公司信用状况
- [ ] 资本结构是否使用目标结构
- [ ] 是否进行了敏感性分析
- [ ] WACC是否在行业合理区间内

---

## 七、常见错误

1. **使用账面值计算权重**：应使用股权市值
2. **忽略税盾效应**：债务成本要乘以(1-T)
3. **直接使用原始Beta**：应根据目标资本结构调整
4. **ERP使用错误**：A股ERP通常高于美股
5. **忽略规模溢价**：小公司需额外加1-4%
6. **资本结构不一致**：Beta对应的D/E与WACC权重应匹配
