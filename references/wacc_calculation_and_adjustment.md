# WACC计算与调整详解

> WACC（加权平均资本成本）是DCF估值的核心参数，直接影响估值结果。

---

## 一、核心公式

### WACC公式
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - T)

E = 股权市值，D = 债务市值，V = E + D
Re = 股权成本，Rd = 债务成本，T = 所得税率
```

### 股权成本（CAPM）
```
Re = Rf + β × ERP + SP + α

Rf = 无风险利率
β = 股权Beta
ERP = 股权风险溢价
SP = 规模溢价
α = 公司特定风险溢价（可选）
```

---

## 二、参数获取

### 2.1 无风险利率（Rf）

| 市场 | 基准 | 参考值 |
|------|------|--------|
| A股 | 10年期国债 | 2.5-3.0% |
| 港股 | 10年期美债/港债 | 4.0-4.5% |
| 美股 | 10年期美债 | 4.0-4.5% |

**注意**：Rf须与现金流币种匹配

### 2.2 股权风险溢价（ERP）

| 市场 | ERP参考 |
|------|---------|
| A股 | 6% |
| 港股 | 6.5-7% |
| 美股 | 4.5-5% |
| 新兴市场 | 7-8% |

### 2.3 Beta系数

**去杠杆（获取资产Beta）**：
```
βu = βL / [1 + (1-T) × (D/E)]
```

**再杠杆（按目标资本结构）**：
```
βL = βu × [1 + (1-T) × (D/E)]
```

**示例**：行业βu=1.1，目标D/E=0.5，T=25%
→ βL = 1.1 × [1 + 0.75 × 0.5] = 1.51

### 2.4 规模溢价（SP）

| 市值规模 | 规模溢价 |
|----------|----------|
| >1000亿 | 0% |
| 200-1000亿 | +0.5-1.0% |
| 50-200亿 | +1.0-2.0% |
| <50亿 | +2.0-4.0% |

### 2.5 债务成本（Rd）

| 信用评级 | 利差 | Rd参考（Rf=3%） |
|----------|------|-----------------|
| AAA | 0.5-1.0% | 3.5-4.0% |
| AA+ | 1.0-1.5% | 4.0-4.5% |
| AA | 1.5-2.0% | 4.5-5.0% |
| AA- | 2.0-3.0% | 5.0-6.0% |
| A+ | 3.0-4.0% | 6.0-7.0% |

### 2.6 资本结构

**使用优先级**：
1. 目标资本结构
2. 行业平均资本结构
3. 当前资本结构

**注意**：
- 股权E用市值，非账面值
- 考虑租赁负债资本化

## 三、特殊情况调整

### 净现金公司
```
- 方案一：WACC = Re（全股权）
- 方案二：用行业平均资本结构
```

### 跨国公司
```
- 多币种现金流分别用对应Rf
- 新兴市场子公司加国家风险溢价
```

### 周期性行业
```
- 周期底部：用较高β（历史高位）
- 周期顶部：用正常化β
```

### 高增长公司
```
分阶段WACC：
- 高增长期：12-15%（高风险）
- 稳定期：8-10%（接近成熟公司）
```

## 六、常见错误

1. **用账面值算权重**：应用股权市值
2. **忽略税盾**：Rd要乘(1-T)
3. **直接用原始Beta**：应做去杠杆/再杠杆调整
4. **ERP用错**：A股ERP高于美股
5. **忽略规模溢价**：小公司需加1-4%
6. **资本结构不一致**：Beta的D/E要与WACC权重匹配
