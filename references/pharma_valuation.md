# 医药行业估值详解

> 医药行业细分众多，创新药、仿制药、医疗器械、CXO、流通等子行业估值逻辑差异显著。
> 本文档详细阐述各医药子行业的估值方法和关键要点。

---

## 一、医药行业分类与估值方法速查

| 子行业 | 首选方法 | 辅助方法 | 核心指标 |
|--------|----------|----------|----------|
| **创新药（Biotech）** | rNPV、管线估值 | PS、对标估值 | 管线阶段、适应症空间、成功概率 |
| **创新药（Big Pharma）** | DCF、SOTP | PE | 成熟产品+在研管线 |
| **仿制药** | PE、EV/EBITDA | DCF | 一致性评价、集采影响 |
| **医疗器械-高值耗材** | PE、PS | DCF | 集采、国产替代 |
| **医疗器械-设备** | PE、EV/EBITDA | DCF | 装机量、服务收入 |
| **CXO** | PE、EV/EBITDA | DCF | 订单、产能利用率 |
| **医药流通** | PE、EV/EBITDA | — | 毛利率、周转率 |
| **中药** | PE | DCF | 品牌、独家品种 |

## 二、创新药估值（核心重点）

### 2.1 为什么创新药需要特殊估值方法

- **高研发投入，长期亏损**：传统PE不适用
- **二元结果**：药品要么成功上市（巨大回报），要么失败（归零）
- **时间价值**：从研发到上市通常10-15年
- **专利悬崖**：专利到期后收入断崖式下跌

### 2.2 rNPV（风险调整净现值）方法

**核心思想**：对每个在研管线分别估值，考虑成功概率后加总

**计算公式**：
```
单个管线价值 = Σ(成功概率 × 各阶段现金流 / (1+r)^t)

其中现金流包括：
- 研发投入（负）
- 峰值销售收入
- 里程碑收款（如有）
- 专利到期后的收入衰减
```

**完整公式展开**：
```
管线NPV = -研发成本现值 + 成功概率 × (峰值销售 × 净利率 × 销售年限折现因子)

公司价值 = Σ(各管线NPV) + 现有产品价值 + 净现金
```

### 2.3 临床阶段成功概率参考

| 阶段 | 成功进入下阶段概率 | 累计成功概率（至上市） | 说明 |
|------|-------------------|----------------------|------|
| **临床前** | 40-50% | 5-10% | 高度不确定 |
| **Phase I** | 60-70% | 10-20% | 安全性验证 |
| **Phase II** | 35-45% | 20-35% | 关键概念验证 |
| **Phase III** | 60-70% | 50-70% | 大样本有效性 |
| **NDA/BLA** | 85-95% | 85-95% | 审批阶段 |
| **已上市** | — | 95%+ | 商业化风险 |

**按适应症调整**：

| 适应症 | 概率调整 | 说明 |
|--------|----------|------|
| 肿瘤 | 基准 | 数据丰富 |
| 罕见病 | +10-20% | 监管支持 |
| CNS（神经） | -10-20% | 临床难度高 |
| 代谢/心血管 | -5-10% | 终点难达 |
| 抗感染 | +5-10% | 相对成熟 |

### 2.4 峰值销售预测方法

**方法一：流行病学法**
```
峰值销售 = 患者人数 × 渗透率 × 年治疗费用 × 市占率

示例：PD-1抑制剂用于肺癌
- 中国肺癌新发患者：80万/年
- 适用人群比例：30%（24万）
- PD-1渗透率：50%（12万）
- 年治疗费用：10万元
- 预期市占率：15%
- 峰值销售 = 12万 × 10万 × 15% = 18亿元
```

**方法二：对标法**
```
峰值销售 = 对标药物峰值销售 × 调整系数

调整系数考虑：
- 疗效差异（更优+20-50%，更差-20-50%）
- 定价差异
- 竞争格局（更拥挤-20-40%）
- 市场差异（中国vs全球）
```

### 2.5 创新药DCF参数设定

| 参数 | 早期（临床前-II期） | 中后期（III期-上市） | 成熟期 |
|------|-------------------|---------------------|--------|
| **折现率** | 15-20% | 12-15% | 10-12% |
| **预测期** | 至专利到期后5年 | 至专利到期后5年 | 10年 |
| **永续增长率** | 不适用 | 不适用 | 0-2% |
| **专利到期后衰减** | -50%/年（首年） | -50%/年（首年） | -30%/年 |

### 2.6 创新药估值案例

**案例：某Biotech公司，3条核心管线**

| 管线 | 适应症 | 阶段 | 成功概率 | 峰值销售 | 净利率 | NPV |
|------|--------|------|----------|----------|--------|-----|
| A | 肺癌 | III期 | 55% | 30亿 | 25% | 22亿 |
| B | 乳腺癌 | II期 | 25% | 20亿 | 25% | 8亿 |
| C | 肝癌 | I期 | 12% | 15亿 | 25% | 3亿 |

```
管线总价值 = 22 + 8 + 3 = 33亿
净现金 = 10亿
公司价值 = 43亿
```

## 三、仿制药估值

### 3.1 仿制药估值特点

- **盈利可预测**：产品成熟，收入利润相对稳定
- **集采冲击**：中标降价60-90%，毛利率大幅压缩
- **一致性评价**：是参与集采的前提条件

### 3.2 估值方法

**主方法：PE估值**
```
目标价 = EPS × 目标PE

目标PE选取：
- 龙头、转型创新：20-30x
- 一般仿制药：12-18x
- 集采压力大：8-12x
```

**辅助方法：EV/EBITDA**
```
目标EV = EBITDA × 目标倍数
股权价值 = EV - 净债务

倍数参考：8-15x
```

### 3.3 集采影响评估

**集采降价影响测算**：
```
收入影响 = 集采品种原收入 × (1 - 中标降幅) × 中标概率
         + 集采品种原收入 × 丢标后残余率 × (1 - 中标概率)

利润影响 = 收入影响 - 成本节约（销售费用下降等）
```

**集采后估值调整**：

| 情景 | PE调整 | 说明 |
|------|--------|------|
| 核心产品未集采 | 维持 | 观察期 |
| 集采中标，降幅<50% | -10-20% | 以价换量 |
| 集采中标，降幅>70% | -30-50% | 利润大幅压缩 |
| 集采丢标 | -50%+ | 收入断崖 |
| 转型创新成功 | +20-50% | 估值重构 |

### 3.4 仿制药关键指标

| 指标 | 优秀 | 良好 | 警惕 |
|------|------|------|------|
| **毛利率** | >60% | 50-60% | <50% |
| **研发费用率** | >10% | 5-10% | <5% |
| **一致性评价通过数** | >30个 | 15-30个 | <15个 |
| **在研创新药管线** | >5个 | 1-5个 | 0 |

## 四、医疗器械估值

### 4.1 高值耗材

**估值特点**：
- 国产替代逻辑
- 集采降价风险
- 技术迭代快

**估值方法**：PE为主，PS辅助（亏损期）

**PE参考**：
| 类型 | PE区间 | 说明 |
|------|--------|------|
| 心血管介入（龙头） | 30-50x | 国产替代空间大 |
| 骨科植入（龙头） | 25-40x | 集采已落地 |
| 眼科（龙头） | 35-50x | 渗透率低 |
| 一般耗材 | 15-25x | 竞争充分 |

### 4.2 医疗设备

**估值特点**：
- 装机量驱动
- 服务收入占比提升
- 出海逻辑

**估值方法**：PE、EV/EBITDA

**PE参考**：
| 类型 | PE区间 | 说明 |
|------|--------|------|
| 影像设备（CT、MRI） | 25-40x | 国产替代 |
| 超声 | 25-35x | 分级诊疗受益 |
| 内镜 | 30-45x | 渗透率低 |
| IVD（化学发光） | 25-40x | 国产替代 |

### 4.3 器械估值关键指标

| 指标 | 说明 | 优秀基准 |
|------|------|----------|
| **毛利率** | 体现技术壁垒 | >60% |
| **研发费用率** | 创新投入 | >10% |
| **海外收入占比** | 出海能力 | >20% |
| **服务收入占比** | 商业模式粘性 | >15% |

## 五、CXO估值

### 5.1 CXO估值特点

- **订单驱动**：在手订单是前瞻指标
- **产能利用率**：影响盈利弹性
- **客户结构**：大客户占比风险

### 5.2 估值方法

**主方法：PE**
```
目标PE = 行业PE × 增长率溢价 × 客户结构调整

参考区间：
- 龙头CXO：35-60x
- 二线CXO：25-40x
- 临床CRO：30-50x
- CDMO：40-70x
```

**辅助方法：EV/EBITDA**
```
参考区间：15-30x
```

### 5.3 CXO关键指标

| 指标 | 定义 | 优秀 | 良好 | 警惕 |
|------|------|------|------|------|
| **订单增速** | 新签订单同比 | >30% | 15-30% | <10% |
| **在手订单/收入** | 订单覆盖度 | >2x | 1.5-2x | <1.5x |
| **产能利用率** | 实际/设计产能 | >80% | 60-80% | <60% |
| **大客户占比** | 前5大客户收入占比 | <30% | 30-50% | >50% |
| **海外收入占比** | 全球化能力 | >50% | 30-50% | <30% |

## 六、医药估值检查清单

### 6.1 创新药

- [ ] 管线阶段和成功概率假设是否合理
- [ ] 峰值销售预测依据（患者数、渗透率、定价、市占率）

- [ ] 专利到期时间和仿制药冲击

### 6.2 仿制药

- [ ] 集采涉及品种和降价幅度测算
- [ ] 一致性评价进度
- [ ] 转型创新进展

### 6.3 医疗器械

- [ ] 国产替代空间测算
- [ ] 集采政策影响（高值耗材）
- [ ] 技术迭代风险
- [ ] 海外市场进展

- [ ] 在手订单充足度
- [ ] 大客户集中度风险
- [ ] 产能扩张计划和资本开支
- [ ] 海外产能布局

## 七、医药估值常见陷阱

1. **过度乐观的管线成功概率**：早期管线实际成功率远低于临床报告
2. **忽视竞争格局**：同靶点、同适应症竞品会分流市场
3. **峰值销售外推过长**：专利到期后收入快速下滑
4. **集采影响低估**：降价幅度和范围往往超预期
5. **研发投入资本化**：部分公司将研发费用资本化美化利润
6. **忽视商业化能力**：有好产品但缺乏销售能力的公司需折价
