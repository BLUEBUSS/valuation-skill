# 周期性行业估值详解

> 周期性行业（能源、资源、制造）的盈利随宏观经济和大宗商品波动，估值需特别考虑周期位置。

---

## 一、周期性行业特征

| 特征 | 说明 |
|------|------|
| **盈利波动大** | 顶部利润可能是底部的5-10倍 |
| **PE失真** | 周期顶部PE低（利润高），底部PE高（利润低） |
| **均值回归** | 价格和盈利长期围绕均值波动 |
| **资本密集** | 固定资产投入大，折旧高 |

**典型行业**：煤炭、钢铁、有色金属、石油天然气、化工、航运、造纸

---

## 二、周期位置判断

### 2.1 判断指标

| 指标 | 周期顶部信号 | 周期底部信号 |
|------|--------------|--------------|
| **产品价格** | 价格>历史均值+1标准差 | 价格<历史均值-1标准差 |
| **产能利用率** | >90% | <70% |
| **库存水平** | 低库存、供不应求 | 高库存、供过于求 |
| **行业资本开支** | 大规模扩产 | 资本开支收缩 |
| **企业盈利** | ROE>20%、毛利率新高 | 亏损面扩大、现金流紧张 |
| **行业情绪** | 乐观、新玩家进入 | 悲观、破产重组 |

### 2.2 周期阶段划分

```
复苏期 → 扩张期 → 顶部 → 收缩期 → 底部 → 复苏期
```

| 阶段 | 特征 | 估值策略 |
|------|------|----------|
| **底部** | 亏损或微利，产能出清 | 用PB，布局时机 |
| **复苏期** | 价格回升，盈利改善 | PB→PE切换 |
| **扩张期** | 量价齐升，利润高增 | PE、EV/EBITDA |
| **顶部** | 盈利高峰，估值要谨慎 | 用PB，警惕PE陷阱 |
| **收缩期** | 价格下跌，盈利恶化 | 规避，或用正常化PE |

---

## 三、估值方法选择

### 3.1 周期不同阶段的方法

| 周期位置 | 首选方法 | 禁用 | 说明 |
|----------|----------|------|------|
| **底部** | PB、EV/EBITDA | 当期PE | 盈利失真，用资产价值 |
| **复苏期** | 正常化PE、PB | — | 盈利恢复中 |
| **扩张/顶部** | PB、正常化PE | 当期PE | 避免高估 |
| **收缩期** | PB、EV/EBITDA | PE | 盈利下滑 |

### 3.2 核心估值方法

#### 方法一：正常化盈利PE

**原理**：用周期中枢盈利而非当期盈利

```
正常化净利润 = 过去5-7年平均净利润
            或 = 正常化价格 × 正常销量 × 正常化利润率

正常化PE = 当前市值 / 正常化净利润
目标价 = 正常化EPS × 合理PE（历史中枢）
```

#### 方法二：PB估值

**原理**：资产价值相对稳定

```
合理PB = f(正常化ROE)
- 正常化ROE 8-10% → PB 0.8-1.0x
- 正常化ROE 10-15% → PB 1.0-1.5x
- 正常化ROE >15% → PB 1.5-2.0x
```

#### 方法三：EV/EBITDA

**原理**：消除折旧政策和资本结构差异

```
合理EV/EBITDA参考：
- 能源（油气）：3-6x
- 煤炭：3-5x
- 钢铁：4-6x
- 有色金属：5-8x
- 化工：5-8x
```

## 四、能源（油气）估值要点

### 4.1 特殊方法：储量估值

```
NAV = Σ(证实储量 × 单位储量价值) + 净现金 - 净负债

单位储量价值参考：
- 石油：$5-15/桶（取决于开采成本和油价假设）
- 天然气：$1-3/千立方英尺
```

### 4.2 油价假设

| 情景 | 布伦特油价假设 | 说明 |
|------|----------------|------|
| 保守 | $60-70/桶 | 长期均衡价 |
| 中性 | $70-80/桶 | 当前中枢 |
| 乐观 | $80-100/桶 | 供应紧张 |

### 4.3 关键指标

| 指标 | 说明 | 健康标准 |
|------|------|----------|
| **储量替代率** | 新增储量/当年产量 | >100% |
| **储采比** | 证实储量/年产量 | >10年 |
| **完全成本** | 含折旧的单位成本 | <$50/桶 |
| **现金成本** | 不含折旧的单位成本 | <$30/桶 |

## 五、金属矿业估值要点

### 5.1 储量估值

```
NAV = Σ(矿山储量 × 金属价格 × 回收率 - 开采成本) × 折现因子

关键变量：
- 储量品位（影响回收率和成本）
- 金属价格假设
- 开采成本（露天 vs 地下）
- 矿山寿命
```

### 5.2 金属价格假设原则

- **短期**：参考期货价格曲线
- **长期**：用历史均价或成本曲线支撑价
- **敏感性**：必须做价格±20%的敏感性分析

### 5.3 主要金属长期价格参考

| 金属 | 长期价格参考 | 说明 |
|------|--------------|------|
| 铜 | $7,000-8,500/吨 | 供需平衡价 |
| 铝 | $2,000-2,500/吨 | 成本支撑 |
| 锂 | 波动大，用成本曲线 | 新兴市场 |
| 铁矿石 | $80-100/吨 | 长期均衡 |

## 六、制造业周期估值

### 6.1 周期性制造业

**典型行业**：钢铁、化工、造纸、水泥、玻璃

**估值要点**：
- 判断产能周期位置（扩张/出清）
- 使用3-5年平均盈利
- PB作为底部安全边际

### 6.2 PB-ROE框架

| 正常化ROE | 建议PB |
|-----------|--------|
| <8% | 0.5-0.8x |
| 8-12% | 0.8-1.2x |
| 12-18% | 1.2-1.8x |
| >18% | 1.8-2.5x |

### 6.3 产能周期判断

| 信号 | 产能扩张期 | 产能出清期 |
|------|------------|------------|
| 行业资本开支 | 大幅增长 | 持续收缩 |
| 产能利用率 | 下降趋势 | 触底回升 |
| 行业集中度 | 分散 | 提升 |
| 龙头策略 | 抢份额 | 稳价格 |

## 七、周期股估值陷阱

### 7.1 PE陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| **低PE陷阱** | 周期顶部PE低，但盈利即将下滑 | 用正常化PE或PB |
| **高PE陷阱** | 周期底部PE高，但盈利即将恢复 | 看PB和复苏信号 |

### 7.2 判断口诀

```
周期顶部：PE低、PB高、盈利好 → 警惕，不是便宜
周期底部：PE高、PB低、盈利差 → 可能是机会
```

### 7.3 估值检查要点

- [ ] 当前处于周期什么位置？
- [ ] 产品价格相对历史均值的位置？
- [ ] 产能利用率水平？
- [ ] 是否使用了正常化盈利？
- [ ] PB是否在历史区间内？
- [ ] 是否做了价格敏感性分析？

---

## 八、估值输出要求

周期性行业估值报告必须包含：

1. **周期位置判断**：当前处于周期哪个阶段，依据是什么
2. **正常化盈利假设**：如何计算正常化盈利，假设的产品价格
3. **PB安全边际**：当前PB vs 历史底部PB
4. **价格敏感性**：产品价格±20%对估值的影响
5. **周期催化剂**：什么因素会驱动周期反转
