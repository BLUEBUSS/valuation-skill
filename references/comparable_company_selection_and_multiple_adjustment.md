# 可比公司选择与乘数调整详解

> 相对估值法的核心在于可比公司的选择。选错可比公司会导致估值严重失真。
> 本文档提供严格的可比公司筛选规则和乘数调整方法。

---

## 一、可比公司筛选规则（三大核心条件）

### 1.1 核心业务相似性（最重要）

**原则**：主营业务收入来源和商业模式必须相似

**判断标准**：
- 核心收入来源的业务模式是否相同
- 主要客户群体是否相似
- 盈利模式是否一致（广告、订阅、交易佣金等）

**正确示例**：

| 目标公司 | 核心业务 | 可比公司 | 理由 |
|----------|----------|----------|------|
| 腾讯-游戏 | 游戏 | 网易、米哈游、EA、动视暴雪 | 同为游戏收入 |
| 腾讯-社交广告 | 广告 | Meta、快手、B站 | 同为社交/内容广告 |
| 腾讯-音乐 | 音乐流媒体 | Spotify、网易云音乐 | 同为音乐订阅 |
| 贵州茅台 | 高端白酒 | 五粮液、泸州老窖 | 同为高端白酒 |
| 宁德时代 | 动力电池 | LG新能源、比亚迪电池、亿纬锂能 | 同为电池制造 |

**错误示例**：

| 目标公司 | 错误可比 | 问题 |
|----------|----------|------|
| 腾讯 | 亚马逊 | 亚马逊以电商+云为主，业务逻辑完全不同 |
| 腾讯 | 阿里巴巴 | 阿里以电商为主，虽同为互联网但核心业务不同 |
| 茅台 | 青岛啤酒 | 啤酒vs白酒，消费场景和定价逻辑不同 |
| 宁德时代 | 特斯拉 | 整车vs电池，产业链位置不同 |

### 1.2 同一市场或可比市场

**原则**：优先选择同一上市市场的可比公司

**规则**：

| 目标公司上市地 | 可比公司优先级 |
|----------------|----------------|
| A股 | A股 > 港股 > 美股 |
| 港股 | 港股 ≈ A股 > 美股 |
| 美股 | 美股 > 港股 > A股 |

**跨市场调整**：
- 不同市场存在估值差异（流动性溢价、投资者结构）
- 使用美股可比时，需考虑A股/港股通常有10-30%折价
- 使用A股可比对标港股时，需考虑A股可能有流动性溢价

### 1.3 规模可比性

**原则**：规模差异过大会导致估值不可比

**量化标准**：

| 指标 | 可比区间 | 说明 |
|------|----------|------|
| 营收规模 | 0.3x - 3x | 目标公司营收在可比公司的0.3-3倍区间 |
| 市值规模 | 0.2x - 5x | 市值差异不宜过大 |
| 增长率差异 | ±15个百分点 | 增长阶段应相近 |

**规模调整**：
- 小公司通常享有成长溢价（更高PE）
- 大公司通常享有稳定性溢价（更低风险折价）
- 龙头公司可能享有品牌/垄断溢价

## 二、大型多元化公司：必须使用SOTP分部估值

### 2.1 适用范围

**以下类型公司必须采用SOTP（Sum of the Parts）分部估值**：

- **大型互联网平台**：腾讯、阿里、字节跳动、美团
- **科技巨头**：苹果、微软、谷歌、亚马逊、Meta、英伟达、特斯拉
- **多元化集团**：伯克希尔、复星、中信
- **业务跨度大的公司**：任何主营业务超过3个且各占收入20%以上的公司

### 2.2 SOTP估值方法

**步骤**：

```
Step 1: 拆分业务板块
        - 按收入/利润贡献拆分
        - 每个板块应有独立的财务数据

Step 2: 为每个板块选择对应的可比公司
        - 游戏板块 → 纯游戏公司
        - 广告板块 → 纯广告公司
        - 云服务板块 → 纯云公司

Step 3: 分别估值
        - 每个板块用对应行业的方法和倍数
        - 汇总得到企业总价值

Step 4: 调整
        - 加：净现金、投资性资产
        - 减：控股公司折价（通常10-20%）
        - 减：净债务
```

### 2.3 腾讯SOTP估值示例

| 业务板块 | 收入占比 | 估值方法 | 可比公司 |
|----------|----------|----------|----------|
| 游戏 | 30% | PE/EV-EBITDA | 网易、EA、动视暴雪 |
| 社交广告 | 20% | PE/EV-EBITDA | Meta、快手 |
| 金融科技 | 25% | PE | 蚂蚁（参考）、PayPal |
| 云服务 | 10% | EV/Revenue | 阿里云、AWS |
| 音乐 | 5% | EV/Revenue | Spotify、网易云 |
| 投资组合 | — | NAV | 按持股市值计算 |

```
腾讯价值 = 游戏估值 + 广告估值 + 金融估值 + 云估值 + 音乐估值 + 投资组合NAV - 控股折价
```

## 三、可比公司数量和选择

### 3.1 数量要求

| 情景 | 最少数量 | 建议数量 |
|------|----------|----------|
| 单一业务公司 | 3家 | 5-8家 |
| SOTP各板块 | 2家 | 3-5家 |

### 3.2 异常值处理

**剔除规则**：
- PE/PB/PS超过可比公司中位数2倍以上 → 考虑剔除
- PE/PB/PS低于可比公司中位数50%以下 → 考虑剔除
- 或使用中位数而非平均值（更稳健）

### 3.3 选择优先级

```
1. 业务模式高度相似（权重最高）
2. 同一上市市场
3. 规模相近
4. 增长阶段相近
5. 财务结构相近
```

## 四、乘数调整方法

### 4.1 增长率调整

```
目标PE = 可比PE × (1 + 增长率差异调整)

调整系数参考：
- 目标增长率高于可比10个百分点 → +15-25%
- 目标增长率低于可比10个百分点 → -15-25%
```

### 4.2 盈利能力调整

```
目标PS = 可比PS × (目标毛利率 / 可比毛利率)

示例：
- 目标公司毛利率50%，可比公司毛利率40%
- 调整系数 = 50%/40% = 1.25
- 目标PS应为可比PS的1.25倍
```

### 4.3 风险调整

| 风险因素 | 折价幅度 |
|----------|----------|
| 客户集中度高（前5大>50%） | -10-20% |
| 政策风险高 | -10-30% |
| 竞争格局恶化 | -10-20% |
| 管理层不稳定 | -5-15% |

## 五、可比公司选择检查清单

- [ ] 核心业务模式是否相似？（最重要）
- [ ] 是否优先选择同一市场的公司？
- [ ] 规模是否在0.3x-3x区间？
- [ ] 增长阶段是否相近？
- [ ] 是否剔除了异常值？
- [ ] 大型多元化公司是否采用了SOTP？
- [ ] 可比公司数量是否≥3家？

## 六、常见错误

1. **业务逻辑不同强行对比**
   - ❌ 腾讯 vs 亚马逊（社交游戏 vs 电商云）
   - ✅ 腾讯游戏 vs 网易游戏

2. **忽略市场差异**
   - ❌ 直接用美股PE套A股公司
   - ✅ 考虑市场溢价/折价调整

3. **对大型集团用单一倍数**
   - ❌ 对腾讯整体用一个PE
   - ✅ 分部估值后加总

4. **可比公司数量不足**
   - ❌ 只选1-2家可比
   - ✅ 至少3家，建议5-8家

5. **不剔除异常值**
   - ❌ 直接用平均值（含异常值）
   - ✅ 用中位数或剔除异常后平均
