# 估值案例框架

> 本文档提供估值案例的方法选择逻辑和输出框架，而非详细计算过程。

---

## 案例一：成熟消费公司（PE + DCF）

### 方法选择逻辑
- 成熟期 + 消费行业 → 首选PE
- 现金流稳定可预测 → DCF可用
- 盈利稳定 → PE有效
- **选定**：PE为主，DCF交叉验证

### PE估值框架
1. **选可比公司**：同行业、同商业模式、规模相近（3-5家）
2. **取可比PE中位数**
3. **调整溢价/折价**：
   - 增长率高于可比 → +5-10%
   - 净利率高于可比 → +3-5%
   - 龙头地位 → +5%
4. **计算**：目标价 = EPS × 目标PE

### DCF估值框架
| 假设项 | 成熟消费参考 |
|--------|--------------|
| 预测期 | 10年 |
| 营收增速 | 前5年10-15%，后5年5-8% |
| 净利率 | 维持或小幅提升 |
| WACC | 8-10%（成熟消费龙头） |
| 永续增长率 | 2-3% |

### 结论框架
```
PE估值：XX元（权重50%）
DCF估值：XX元（权重50%）
综合目标价：XX元
当前股价：XX元 → 涨幅XX% → 评级：买入/增持/持有
```

## 案例二：成长型SaaS公司（EV/ARR + 远期DCF）

### 方法选择逻辑
- 亏损 → **PE不适用**
- 高增长SaaS → EV/ARR、PS
- 未来可盈利 → 远期DCF验证
- **选定**：EV/ARR为主，远期DCF验证

### EV/ARR估值框架
1. **选可比公司**：业务模式、增长阶段相似的SaaS
2. **取可比EV/ARR中位数**
3. **调整溢价/折价**：
   - ARR增速高于可比10pct → +10-15%
   - NRR高于可比10pct → +10%
   - 毛利率低于可比 → -2-5%
4. **计算**：股权价值 = ARR × 目标倍数 + 净现金

### 远期DCF框架
| 假设项 | 成长期SaaS参考 |
|--------|----------------|
| 预测期 | 10年（至盈利稳定） |
| ARR增速 | 从50%逐年递减至15% |
| 盈利拐点 | 第3-5年 |
| 稳态净利率 | 15-20% |
| WACC | 12-15%（高风险） |
| 永续增长率 | 3-4% |

### 关键洞察
- 若DCF估值 **显著低于** EV/ARR估值 → 市场预期激进，需关注增长持续性
- 若DCF估值 **高于** 当前市值 → 可能被低估

## 案例三：银行（PB + DDM）

### 方法选择逻辑
- 银行不用DCF（利息收支难分离、高杠杆）
- PB-ROE框架是银行估值核心
- 稳定分红 → DDM辅助
- **选定**：PB为主（60%），DDM辅助（40%）

### PB估值框架
1. **PB-ROE对应**：
   - ROE < 10% → PB 0.6-0.8x
   - ROE 10-12% → PB 0.8-1.0x
   - ROE 12-15% → PB 1.0-1.3x
   - ROE > 15% → PB 1.3-1.6x

2. **调整因素**：
   - 不良率低（<1%）→ +5%
   - 拨备覆盖率高（>250%）→ +5%
   - 区域优质（长三角、珠三角）→ +5%

3. **计算**：目标价 = BVPS × 目标PB

### DDM估值框架
| 假设项 | 银行参考 |
|--------|----------|
| 当前股息 | EPS × 分红率（通常30%） |
| 股息增长率 | 近期5-8%，永续3-4% |
| 股权成本 | 9-11% |

## 案例四：创新药Biotech（rNPV管线估值）

### 方法选择逻辑
- 亏损+研发驱动 → **PE不适用**
- 管线价值是核心 → rNPV
- **选定**：rNPV管线估值

### rNPV估值框架

**Step 1：逐条管线估值**

| 管线要素 | 说明 |
|----------|------|
| 临床阶段 | 决定成功概率 |
| 适应症空间 | 决定峰值销售 |
| 竞争格局 | 调整市占率 |

**成功概率参考**：
- I期：10-15%
- II期：25-30%
- III期：50-60%
- 已上市：90%+

**峰值销售计算**：
```
峰值销售 = 患者人数 × 渗透率 × 年治疗费用 × 市占率
```

**单管线rNPV**：
```
rNPV = NPV（假设成功）× 成功概率
```

**Step 2：公司价值汇总**
```
公司价值 = Σ各管线rNPV + 现有业务价值 + 净现金 - 未来研发投入
```

### 关键洞察
- 若市值 **远高于** rNPV → 市场隐含更高成功概率或峰值假设
- 关注临床数据读出的催化剂时点
- 需做成功概率±10%的敏感性分析

## 案例速查

| 公司类型 | 方法组合 | 关键判断 |
|----------|----------|----------|
| 成熟消费/制造 | PE + DCF | 盈利稳定→PE有效 |
| 成长期亏损科技 | EV/ARR + 远期DCF | 亏损→禁用PE |
| 银行 | PB + DDM | PB-ROE框架 |
| 保险 | P/EV + NBV倍数 | 内含价值法 |
| 创新药Biotech | rNPV | 管线分部估值 |
| 房地产 | NAV + PB | 项目货值折现 |
| 周期性行业 | PB + 正常化PE | 避免PE陷阱 |
