## 案例一：成熟消费公司（PE + DCF）

### 1.1 公司概况

**目标公司**：某高端白酒公司（假设）
- **行业**：消费-白酒
- **生命周期**：成熟期
- **特点**：品牌力强、毛利率高、现金流稳定

**基础财务数据**：

| 指标 | 数值 |
|------|------|
| 当前股价 | 180元 |
| 总股本 | 12.6亿股 |
| 市值 | 2,268亿元 |
| 最近年度营收 | 650亿元 |
| 最近年度净利润 | 260亿元 |
| 每股收益（EPS） | 20.6元 |
| 每股净资产 | 85元 |
| 净利率 | 40% |
| ROE | 24% |
| 有息负债 | 20亿元 |
| 现金 | 400亿元 |

### 1.2 估值方法选择

**依据**：
- 成熟期 + 消费行业 → 首选PE
- 现金流稳定 → DCF可用
- 盈利稳定 → PE有效

**选定方法**：
- 主方法：PE估值
- 辅助方法：DCF估值

### 1.3 PE估值

**Step 1：选择可比公司**

| 可比公司 | 市值 | PE（TTM） | 净利润增速 | 净利率 |
|----------|------|----------|------------|--------|
| 可比A | 2,500亿 | 32x | 15% | 45% |
| 可比B | 800亿 | 28x | 12% | 35% |
| 可比C | 1,200亿 | 30x | 10% | 38% |
| **中位数** | — | **30x** | 12% | 38% |

**Step 2：确定目标PE**

```
目标公司特点 vs 可比公司：
- 增长率：15%（高于中位数12%）→ 溢价5%
- 净利率：40%（高于中位数38%）→ 溢价3%
- 品牌力：龙头地位 → 溢价5%

目标PE = 30x × (1 + 5% + 3% + 5%) = 30x × 1.13 = 34x
保守区间：30-35x
```

**Step 3：计算目标价**

```
乐观：EPS 20.6 × 35x = 721元
中性：EPS 20.6 × 32x = 659元
保守：EPS 20.6 × 30x = 618元

PE估值区间：618-721元
中枢：659元
```

### 1.4 DCF估值

**Step 1：关键假设**

| 假设项 | 数值 | 依据 |
|--------|------|------|
| 预测期 | 10年 | 成熟公司标准 |
| 营收增速（1-5年） | 12%/年 | 历史增速、行业空间 |
| 营收增速（6-10年） | 8%/年 | 逐步放缓 |
| 净利率 | 40%→42% | 规模效应改善 |
| 资本支出/营收 | 3% | 轻资产 |
| 营运资金/营收增量 | 10% | 历史水平 |
| WACC | 9% | 成熟消费龙头 |
| 永续增长率 | 3% | 通胀+微增长 |

**Step 2：现金流预测（简化）**

| 年份 | 营收 | 净利润 | FCF |
|------|------|--------|-----|
| Y1 | 728 | 291 | 275 |
| Y2 | 815 | 334 | 316 |
| Y3 | 913 | 383 | 362 |
| Y4 | 1,023 | 439 | 415 |
| Y5 | 1,146 | 503 | 476 |
| Y6 | 1,238 | 545 | 516 |
| Y7 | 1,337 | 589 | 557 |
| Y8 | 1,444 | 636 | 602 |
| Y9 | 1,559 | 687 | 650 |
| Y10 | 1,684 | 742 | 702 |

**Step 3：计算企业价值**

```
预测期FCF现值 = Σ(FCFt / (1+9%)^t) = 3,200亿元
终值 = 702 × (1+3%) / (9%-3%) = 12,051亿元
终值现值 = 12,051 / (1+9%)^10 = 5,092亿元

企业价值 = 3,200 + 5,092 = 8,292亿元
股权价值 = 8,292 - 20（债务）+ 400（现金）= 8,672亿元
每股价值 = 8,672 / 12.6 = 688元
```

**Step 4：敏感性分析**

| g \ WACC | 8% | 9% | 10% |
|----------|-----|-----|------|
| 2% | 756 | 628 | 532 |
| 3% | 836 | 688 | 578 |
| 4% | 942 | 763 | 634 |

### 1.5 估值结论

**综合估值**：

| 方法 | 结果 | 权重 |
|------|------|------|
| PE估值 | 659元 | 50% |
| DCF估值 | 688元 | 50% |
| **综合目标价** | **674元** | — |

**估值区间**：618-721元

**投资评级**：
```
当前股价：180元
目标价：674元
潜在涨幅：274%
评级：买入
```

> 注：此为假设案例，实际数据需根据真实财务报表调整

## 案例二：成长型科技公司（PS + 远期DCF）

### 2.1 公司概况

**目标公司**：某SaaS软件公司（假设）
- **行业**：科技-SaaS
- **生命周期**：成长期
- **特点**：高增长、尚未盈利、ARR快速增长

**基础数据**：

| 指标 | 数值 |
|------|------|
| 当前股价 | 50元 |
| 总股本 | 4亿股 |
| 市值 | 200亿元 |
| ARR（年度经常性收入） | 15亿元 |
| ARR增速 | 50% |
| NRR（净收入留存率） | 125% |
| 毛利率 | 70% |
| 净利润 | -3亿元（亏损） |
| 现金储备 | 30亿元 |

### 2.2 估值方法选择

**依据**：
- 亏损 → PE不适用
- 高增长SaaS → PS、EV/ARR
- 未来可预测盈利 → 远期DCF

**选定方法**：
- 主方法：EV/ARR
- 辅助方法：远期DCF

### 2.3 EV/ARR估值

**Step 1：选择可比公司**

| 可比公司 | EV/ARR | ARR增速 | NRR | 毛利率 |
|----------|--------|---------|-----|--------|
| 可比A | 12x | 40% | 115% | 75% |
| 可比B | 15x | 55% | 130% | 72% |
| 可比C | 10x | 35% | 110% | 68% |
| **中位数** | **12x** | 40% | 115% | 72% |

**Step 2：确定目标倍数**

```
目标公司 vs 可比中位数：
- ARR增速：50% vs 40% → 溢价15%
- NRR：125% vs 115% → 溢价10%
- 毛利率：70% vs 72% → 折价2%

目标EV/ARR = 12x × (1 + 15% + 10% - 2%) = 12x × 1.23 = 14.8x
取整：15x
保守区间：12-18x
```

**Step 3：计算目标价**

```
EV = ARR × 倍数 = 15亿 × 15x = 225亿
股权价值 = EV + 净现金 = 225 + 30 = 255亿
每股价值 = 255 / 4 = 63.8元

EV/ARR估值区间：
- 保守（12x）：(15×12+30)/4 = 52.5元
- 中性（15x）：63.8元
- 乐观（18x）：(15×18+30)/4 = 75元
```

### 2.4 远期DCF估值

**Step 1：关键假设**

| 假设项 | 数值 | 依据 |
|--------|------|------|
| 预测期 | 10年 | 至盈利稳定 |
| ARR增速 | 50%→45%→...→15% | 逐年递减 |
| 盈利拐点 | 第4年 | 规模效应显现 |
| 稳态净利率 | 20% | 参考成熟SaaS |
| WACC | 14% | 高成长高风险 |
| 永续增长率 | 4% | 行业增长 |

**Step 2：现金流预测（简化）**

| 年份 | ARR | 净利润 | FCF |
|------|-----|--------|-----|
| Y1 | 22.5 | -4 | -5 |
| Y2 | 31.5 | -2 | -3 |
| Y3 | 42.5 | 2 | 1 |
| Y4 | 55.3 | 8 | 7 |
| Y5 | 69.1 | 12 | 11 |
| Y6 | 82.9 | 16 | 15 |
| Y7 | 95.4 | 19 | 18 |
| Y8 | 107.2 | 21 | 20 |
| Y9 | 117.9 | 24 | 22 |
| Y10 | 127.4 | 25 | 24 |

**Step 3：计算企业价值**

```
预测期FCF现值 = 45亿元
终值 = 24 × (1+4%) / (14%-4%) = 250亿元
终值现值 = 250 / (1+14%)^10 = 67亿元

企业价值 = 45 + 67 = 112亿元
股权价值 = 112 + 30 = 142亿元
每股价值 = 142 / 4 = 35.5元
```

### 2.5 估值结论

**综合估值**：

| 方法 | 结果 | 权重 | 说明 |
|------|------|------|------|
| EV/ARR | 63.8元 | 60% | 当前市场认可度 |
| 远期DCF | 35.5元 | 40% | 长期价值锚 |
| **综合目标价** | **52.5元** | — | — |

**投资评级**：
```
当前股价：50元
目标价：52.5元
潜在涨幅：5%
评级：持有

风险提示：
- DCF估值显著低于当前市值，说明市场预期较高
- 需关注增长持续性和盈利拐点
```

## 案例三：银行（PB + DDM）

### 3.1 公司概况

**目标公司**：某股份制银行（假设）
- **行业**：金融-银行
- **特点**：ROE较高、资产质量良好

**基础数据**：

| 指标 | 数值 |
|------|------|
| 当前股价 | 15元 |
| 总股本 | 200亿股 |
| 市值 | 3,000亿元 |
| 净资产 | 4,500亿元 |
| 每股净资产 | 22.5元 |
| 净利润 | 500亿元 |
| EPS | 2.5元 |
| ROE | 11.1% |
| 不良率 | 1.2% |
| 拨备覆盖率 | 280% |
| 股息率 | 5% |
| 分红比例 | 30% |

### 3.2 PB估值

**Step 1：PB-ROE分析**

```
当前PB = 15 / 22.5 = 0.67x
ROE = 11.1%

参考PB-ROE框架：
- ROE 10-12% → 合理PB 0.8-1.0x
- 该行ROE 11.1% → 理论PB约0.9x
```

**Step 2：调整因素**

| 因素 | 情况 | 调整 |
|------|------|------|
| 不良率 | 1.2%（良好） | +5% |
| 拨备覆盖率 | 280%（充足） | +5% |
| 区域分布 | 全国性 | 0% |
| 资本充足 | 达标 | 0% |

```
目标PB = 0.9x × (1 + 10%) = 0.99x ≈ 1.0x
保守区间：0.8-1.1x
```

**Step 3：计算目标价**

```
目标价 = BVPS × 目标PB = 22.5 × 1.0 = 22.5元
区间：18-24.8元
```

### 3.3 DDM估值

**Step 1：关键假设**

| 假设项 | 数值 |
|--------|------|
| D0（当前股息） | 0.75元（EPS×30%） |
| 股息增长率（1-5年） | 8% |
| 股息增长率（永续） | 4% |
| 股权成本（COE） | 10% |

**Step 2：两阶段DDM计算**

```
第一阶段（高增长）：
D1 = 0.75 × 1.08 = 0.81
D2 = 0.81 × 1.08 = 0.87
D3 = 0.87 × 1.08 = 0.94
D4 = 0.94 × 1.08 = 1.02
D5 = 1.02 × 1.08 = 1.10

第一阶段现值 = 0.81/1.1 + 0.87/1.1² + 0.94/1.1³ + 1.02/1.1⁴ + 1.10/1.1⁵
             = 0.74 + 0.72 + 0.71 + 0.70 + 0.68
             = 3.55元

第二阶段（永续）：
P5 = D6 / (r - g) = 1.10 × 1.04 / (10% - 4%) = 19.07元
P5现值 = 19.07 / 1.1⁵ = 11.84元

DDM价值 = 3.55 + 11.84 = 15.39元
```

### 3.4 估值结论

**综合估值**：

| 方法 | 结果 | 权重 |
|------|------|------|
| PB估值 | 22.5元 | 60% |
| DDM估值 | 15.4元 | 40% |
| **综合目标价** | **19.7元** | — |

**投资评级**：
```
当前股价：15元
目标价：19.7元
潜在涨幅：31%
评级：买入
```

## 案例四：创新药公司（rNPV管线估值）

### 4.1 公司概况

**目标公司**：某Biotech公司（假设）
- **行业**：医药-创新药
- **特点**：3条核心管线，尚未盈利

**基础数据**：

| 指标 | 数值 |
|------|------|
| 当前股价 | 30元 |
| 总股本 | 5亿股 |
| 市值 | 150亿元 |
| 营收 | 2亿元（少量收入） |
| 净利润 | -5亿元（亏损） |
| 现金储备 | 25亿元 |
| 研发管线 | 3条核心 |

### 4.2 管线分析

| 管线 | 适应症 | 阶段 | 成功概率 | 预计上市 |
|------|--------|------|----------|----------|
| 产品A | 非小细胞肺癌 | III期 | 55% | 2026 |
| 产品B | HER2乳腺癌 | II期 | 28% | 2028 |
| 产品C | 胃癌 | I期 | 12% | 2030 |

### 4.3 rNPV估值

**产品A估值**：

```
峰值销售假设：
- 患者人数：24万（肺癌×适用比例×渗透率）
- 年治疗费用：8万元
- 市占率：20%
- 峰值销售 = 24万 × 8万 × 20% = 38.4亿元

现金流预测（上市后）：
- 净利率：25%
- 峰值净利润：9.6亿元/年
- 销售期：10年（专利期内）
- 专利到期后衰减：50%/年

NPV（假设上市成功，WACC=12%）= 45亿元
rNPV = 45 × 55% = 24.8亿元
```

**产品B估值**：

```
峰值销售：15亿元
NPV（上市成功）= 20亿元
rNPV = 20 × 28% = 5.6亿元
```

**产品C估值**：

```
峰值销售：12亿元
NPV（上市成功）= 15亿元
rNPV = 15 × 12% = 1.8亿元
```

### 4.4 估值汇总

```
管线价值合计 = 24.8 + 5.6 + 1.8 = 32.2亿元
现有业务价值 = 营收2亿 × 3x PS = 6亿元
净现金 = 25亿元
研发支出储备扣减 = -10亿元（未来3年研发投入）

公司价值 = 32.2 + 6 + 25 - 10 = 53.2亿元
每股价值 = 53.2 / 5 = 10.6元
```

### 4.5 估值结论

**综合估值**：

| 项目 | 价值 |
|------|------|
| 管线A（III期） | 24.8亿 |
| 管线B（II期） | 5.6亿 |
| 管线C（I期） | 1.8亿 |
| 现有业务 | 6亿 |
| 净现金 | 25亿 |
| 研发储备扣减 | -10亿 |
| **公司价值** | **53.2亿** |
| **每股价值** | **10.6元** |

**投资评级**：
```
当前股价：30元
rNPV估值：10.6元
当前市值 vs rNPV：150亿 vs 53亿

结论：
- 当前市值显著高于rNPV估值
- 市场隐含了更高的成功概率或更大的峰值销售假设
- 需关注III期临床数据读出的催化剂

风险提示：
- 临床失败风险
- 商业化能力不确定
- 竞争格局恶化
```
