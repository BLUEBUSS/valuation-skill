# 公司生命周期判断指南

> 正确判断公司所处生命周期是选择估值方法的前提。本文档提供详细的判断标准和边界情况处理。

---

## 一、生命周期四阶段定义

| 阶段 | 核心特征 | 典型公司画像 |
|------|----------|--------------|
| **初创期** | 产品验证中，烧钱获客 | Biotech临床阶段、Pre-revenue SaaS |
| **成长期** | 收入快速增长，亏损或微利 | 新能源车新势力、成长期互联网 |
| **成熟期** | 增长放缓，稳定盈利 | 白酒龙头、银行、公用事业 |
| **衰退期** | 收入下滑，利润收窄 | 传统零售、夕阳行业 |

---

## 二、量化判断标准

### 2.1 主要指标阈值

| 指标 | 初创期 | 成长期 | 成熟期 | 衰退期 |
|------|--------|--------|--------|--------|
| **收入规模** | <1亿或无收入 | 1-100亿 | >100亿 | 不限 |
| **收入增速** | >100%或无意义 | 30-100% | 0-20% | <0% |
| **净利润** | 亏损 | 亏损或微利 | 稳定盈利 | 盈利收窄 |
| **净利率** | 负 | <5%或负 | >10% | 下降趋势 |
| **现金流** | 负 | 负或转正 | 稳定正向 | 正但萎缩 |
| **市场地位** | 未确立 | 份额提升 | 份额稳定 | 份额流失 |

### 2.2 判断流程

```
Step 1: 看盈利状态
        └─ 亏损 → 初创期或成长期
        └─ 盈利 → 成熟期或衰退期

Step 2: 看增长率
        └─ 亏损 + 高增长(>50%) → 成长期
        └─ 亏损 + 低增长/无收入 → 初创期
        └─ 盈利 + 正增长(0-20%) → 成熟期
        └─ 盈利 + 负增长 → 衰退期

Step 3: 看增长趋势
        └─ 增速加速 → 可能处于成长早期
        └─ 增速放缓 → 可能向成熟期过渡
        └─ 增速转负 → 进入衰退期
```

## 三、边界情况处理

### 3.1 成长期 vs 初创期

| 判断要点 | 初创期 | 成长期 |
|----------|--------|--------|
| 收入规模 | <1亿或无收入 | >1亿 |
| 商业模式 | 未验证 | 已跑通 |
| 客户基础 | 试点客户 | 规模化获客 |
| 融资阶段 | 天使/A轮 | B轮及以后 |

**实操建议**：
- 有收入但<1亿，看商业模式是否验证
- SaaS看NRR是否>100%（验证客户粘性）
- Biotech看是否有产品进入III期（验证管线）

### 3.2 成长期 vs 成熟期（关键边界）

| 判断要点 | 成长期 | 成熟期 |
|----------|--------|--------|
| 收入增速 | >30% | <20% |
| 盈利状态 | 亏损或净利率<5% | 净利率>10%稳定 |
| 市场份额 | 仍在提升 | 基本稳定 |
| 资本开支 | 高投入扩张 | 维护性投入 |
| 行业格局 | 未定，玩家多 | 已定，集中度高 |

**实操建议**：
- 增速20-30%区间：看净利率趋势，若净利率提升说明向成熟期过渡
- 若收入增速<20%但仍亏损：可能是"伪成长"或商业模式有问题
- 大行业的龙头可能增速<20%但仍有成长属性（如行业集中度提升）

### 3.3 成熟期 vs 衰退期

| 判断要点 | 成熟期 | 衰退期 |
|----------|--------|--------|
| 收入趋势 | 稳定或微增 | 持续下滑 |
| 利润趋势 | 稳定 | 持续收窄 |
| 市场份额 | 稳定 | 被蚕食 |
| 行业趋势 | 稳态 | 被替代 |

**实操建议**：
- 连续2年以上收入下滑 → 衰退期
- 下滑是行业性的还是公司自身问题？
- 是否有转型/第二曲线？

## 四、特殊情况判断

### 4.1 盈利但高增长

| 情况 | 收入增速>30% + 净利率>10% |
|------|--------------------------|
| 判断 | 成熟期（优质成熟公司） |
| 估值方法 | PE + DCF |
| 示例 | 茅台、腾讯成熟业务 |

**说明**：高增长+高利润说明已过盈利拐点，按成熟期处理，但可给更高PE。

### 4.2 亏损但低增长

| 情况 | 收入增速<20% + 亏损 |
|------|---------------------|
| 判断 | 问题公司，非典型阶段 |
| 估值方法 | PB或清算价值 |
| 风险 | 商业模式可能有缺陷 |

**说明**：低增长还亏损，说明既不是成长也不是成熟，需要特别警惕。

### 4.3 周期性行业公司

| 情况 | 当期亏损但非衰退 |
|------|------------------|
| 判断 | 按周期位置判断，非生命周期 |
| 估值方法 | 参考 cyclical_valuation.md |
| 示例 | 周期底部的煤炭、钢铁 |

**说明**：周期性行业用周期框架，不用生命周期框架。

### 4.4 多业务公司

| 情况 | 各业务处于不同阶段 |
|------|-------------------|
| 判断 | 分业务判断，用SOTP |
| 示例 | 腾讯游戏（成熟）+ 云（成长） |

## 五、生命周期与估值方法映射

| 阶段 | 可用方法 | 禁用方法 | 原因 |
|------|----------|----------|------|
| **初创期** | VC法、可比交易、期权定价 | PE、DCF | 无盈利无现金流 |
| **成长期** | PS、EV/ARR、远期DCF | PE（若亏损） | 收入是主要价值 |
| **成熟期** | PE、EV/EBITDA、DCF、DDM | **PS** | 盈利是主要价值 |
| **衰退期** | PB、清算价值、重置成本 | PS、高PE | 资产是剩余价值 |

### 5.1 成熟期禁用PS的原因

```
成熟期公司已有稳定盈利 → 应该用利润指标
PS只看收入，忽略盈利能力 → 可能严重高估

示例：
公司A：收入100亿，净利润20亿，净利率20%
公司B：收入100亿，净利润5亿，净利率5%

若都用PS=2x估值 → 都值200亿
但PE估值：A=200/20=10x，B=200/5=40x
显然用PS会高估B公司
```

## 六、判断检查清单

- [ ] 当前收入规模是多少？
- [ ] 过去3年收入增速趋势？
- [ ] 当前是否盈利？净利率水平？
- [ ] 现金流是否为正？
- [ ] 市场份额是提升还是稳定/下降？
- [ ] 所在行业是成长行业还是成熟/衰退行业？
- [ ] 是否属于周期性行业？（若是，用周期框架）
- [ ] 是否有多个处于不同阶段的业务？

## 七、阶段过渡信号

### 7.1 成长→成熟信号

| 信号 | 说明 |
|------|------|
| 增速连续放缓 | 从50%→40%→30%→20% |
| 净利率持续提升 | 规模效应显现 |
| 资本开支下降 | 从扩张转向收割 |
| 竞争格局稳定 | 市场份额不再大幅变化 |
| 管理层提高分红 | 现金流充裕无需再投 |

### 7.2 成熟→衰退信号

| 信号 | 说明 |
|------|------|
| 收入连续负增长 | 市场萎缩或份额流失 |
| 利润率持续下滑 | 竞争加剧或成本上升 |
| 客户流失加速 | 被新产品/模式替代 |
| 行业负面事件 | 政策打压、技术替代 |
| 人才流失 | 优秀员工离开 |
