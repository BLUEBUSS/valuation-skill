# SaaS与科技公司估值深度解析

## SaaS公司关键估值指标与乘数
1.  **年度经常性收入**：估值的基础。常用乘数为**企业价值/ARR**。
2.  **净收入留存率**：衡量现有客户收入的增长能力。NRR > 120% 通常被视为优秀，可支撑更高乘数。
3.  **客户获取成本回收期**：CAC / （客户月均贡献毛利 * 12）。回收期 < 12 个月为健康。
4.  **生命周期价值与CAC比率**：LTV/CAC > 3 表明高效的营销投入。

## 针对未盈利SaaS公司的DCF建模要点
由于缺乏当前盈利，DCF模型需基于对**未来单位经济模型**的预测：
1.  **预测收入**：基于现有客户增长（NRR）和新客户获取（销售与营销支出计划）。
2.  **预测毛利率**：随着规模扩大，云基础设施成本占比下降，毛利率应逐步改善。
3.  **预测营业费用**：销售与营销费用短期内可能保持高投入，但效率（CAC回收期）应改善。研发费用可能资本化部分进行调整。
4.  **预测自由现金流拐点**：明确公司预计何时实现正向自由现金流。这是估值的关键转折点。
5.  **WACC设定**：由于高风险，应使用较高的股权成本。参考可比上市SaaS公司的β值并进行杠杆调整。

## 市场法应用：可比公司选择与调整
*   **可比公司**：选择业务模式（如垂直SaaS vs 通用SaaS）、客户规模、增长阶段相似的上市公司。
*   **乘数调整**：
    *   如果目标公司的NRR显著高于可比公司中位数，可在EV/ARR乘数上增加溢价（例如，每高出10个百分点，溢价5%）。
    *   如果目标公司尚未达到现金流盈亏平衡，应对其估值乘数应用折价，或使用基于远期ARR（如2年后）的乘数。

## 综合应用案例：成长期SaaS公司
**步骤**：
1.  **DCF建模**：预测未来5年收入、利润和现金流，直至达到稳定增长状态。使用较高的WACC（例如12-15%）。
2.  **市场法验证**：选取5-8家可比上市公司，计算其EV/ARR、EV/Revenue等乘数区间。根据目标公司的NRR、增长率和毛利率进行调整，得出基于当前和远期ARR的估值区间。
3.  **交叉验证与区间确定**：比较DCF结果与市场法结果。如果差异在20%以内，可取交集或平均值。如果差异较大，需重新审视关键假设（如增长率、WACC、可比公司选择）。
4.  **敏感性分析**：对ARR增长率、CAC、WACC进行敏感性测试，给出估值区间而非单一数字。
