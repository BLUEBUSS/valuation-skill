# 金融行业估值详解

> 金融行业（银行、保险、券商）由于其特殊的商业模式和会计处理，需要采用专门的估值方法。
> 本文档详细阐述各金融子行业的估值逻辑、方法和关键指标。

---

## 一、银行估值

### 1.1 为什么银行不用DCF

- **利息收入与利息支出难以分离**：银行的"成本"（存款利息）和"收入"（贷款利息）都是利息，传统FCF定义不适用
- **资本监管约束**：银行必须维持一定资本充足率，不能自由分配现金流
- **杠杆极高**：银行资产负债率通常>90%，WACC计算意义有限

### 1.2 银行估值核心方法

#### 方法一：PB估值法（主推）

**核心逻辑**：银行价值 = 净资产 × PB倍数

**PB-ROE框架**：
```
理论PB = (ROE - g) / (COE - g)

简化判断：
- ROE > COE → PB应 > 1x
- ROE < COE → PB应 < 1x
- ROE = COE → PB应 = 1x
```

**PB与ROE对应关系**（假设COE=10%，g=3%）：

| ROE | 理论PB | 实际参考PB |
|-----|--------|------------|
| 6% | 0.43x | 0.4-0.5x |
| 8% | 0.71x | 0.6-0.8x |
| 10% | 1.00x | 0.8-1.0x |
| 12% | 1.29x | 1.0-1.2x |
| 15% | 1.71x | 1.3-1.6x |
| 18% | 2.14x | 1.6-2.0x |

**PB调整因素**：

| 因素 | 溢价 | 折价 | 说明 |
|------|------|------|------|
| 资产质量 | 不良率<1% | 不良率>2% | 核心调整因素 |
| 拨备覆盖率 | >300% | <150% | 风险缓冲 |
| 资本充足率 | >15% | <12% | 扩张能力 |
| 存款成本 | 低于同业 | 高于同业 | 负债优势 |
| 区域经济 | 发达地区 | 欠发达地区 | 资产质量相关 |

#### 方法二：DDM股利折现（辅助）

**适用条件**：稳定分红、分红率较高的银行

**计算公式**：
```
股票价值 = D1 / (r - g)

其中：
- D1 = EPS × 分红率
- r = 股权成本（通常8-12%）
- g = 可持续增长率 = ROE × (1-分红率)
```

**示例**：
```
某银行：EPS=2元，分红率50%，ROE=12%，COE=10%
D1 = 2 × 50% = 1元
g = 12% × 50% = 6%
P = 1 / (10% - 6%) = 25元
```

### 1.3 银行估值关键指标

| 指标 | 定义 | 优秀 | 良好 | 警惕 |
|------|------|------|------|------|
| **ROE** | 净利润/平均净资产 | >15% | 10-15% | <10% |
| **NIM** | 净息差 | >2.5% | 2.0-2.5% | <2.0% |
| **不良率** | 不良贷款/总贷款 | <1.0% | 1.0-1.5% | >2.0% |
| **拨备覆盖率** | 拨备/不良贷款 | >300% | 200-300% | <150% |
| **资本充足率** | 资本/风险加权资产 | >15% | 12-15% | <12% |
| **成本收入比** | 营业费用/营业收入 | <30% | 30-35% | >40% |

### 1.4 银行估值要点

1. **关注真实不良**：关注类贷款、逾期90天以上贷款可能是隐藏不良
2. **净息差趋势**：LPR下降周期压缩NIM，需关注负债端成本控制能力
3. **中收占比**：非息收入占比提升意味着更稳定的盈利模式
4. **区域分化**：长三角、珠三角银行通常享有估值溢价

## 二、保险估值

### 2.1 保险估值的特殊性

- **长期合同**：寿险合同期限长达20-30年，当期利润不能反映真实价值
- **精算假设**：利润依赖投资收益率、死亡率、费用率等精算假设
- **会计利润失真**：新业务首年费用高，会计上体现为亏损，但实际创造价值

### 2.2 保险估值核心方法

#### 方法一：内含价值法（PEV）

**核心概念**：

```
内含价值(EV) = 调整后净资产(ANW) + 有效业务价值(VIF)

其中：
- ANW = 净资产 - 要求资本的持有成本
- VIF = 存量保单未来利润的现值
```

**新业务价值(NBV)**：
```
NBV = 新保单首年创造的价值（未来利润现值）
新业务价值率 = NBV / 新单保费（或NBV / 新单APE）
```

**估值公式**：
```
股权价值 = EV + NBV × 倍数

其中NBV倍数反映市场对未来新业务的预期：
- 倍数高 → 看好新业务增长
- 倍数低 → 对新业务增长悲观
```

**P/EV参考**：

| 类型 | P/EV | NBV倍数 | 说明 |
|------|------|---------|------|
| 优质寿险 | 0.9-1.2x | 8-15x | 新业务价值率高、增长稳定 |
| 一般寿险 | 0.6-0.9x | 3-8x | 增长放缓或价值率下降 |
| 困境寿险 | 0.4-0.6x | 0-3x | 新业务萎缩 |
| 财险 | 0.8-1.5x | — | 看综合成本率 |

#### 方法二：P/EV简化法

```
目标价 = 每股EV × 目标P/EV倍数

P/EV倍数选取：
- 看ROE vs COE
- 看新业务增长率
- 看投资收益率稳定性
```

### 2.3 保险估值关键指标

**寿险核心指标**：

| 指标 | 定义 | 优秀 | 良好 | 警惕 |
|------|------|------|------|------|
| **新业务价值率** | NBV/新单APE | >30% | 20-30% | <20% |
| **内含价值增速** | EV同比增长 | >15% | 10-15% | <10% |
| **投资收益率** | 投资收益/投资资产 | >5% | 4-5% | <4% |
| **退保率** | 退保金额/期初准备金 | <3% | 3-5% | >5% |
| **代理人产能** | 人均NBV | 提升 | 稳定 | 下降 |

**财险核心指标**：

| 指标 | 定义 | 优秀 | 良好 | 警惕 |
|------|------|------|------|------|
| **综合成本率** | （赔付+费用）/保费 | <95% | 95-100% | >100% |
| **赔付率** | 赔款/保费 | <65% | 65-70% | >75% |
| **费用率** | 费用/保费 | <30% | 30-35% | >35% |

### 2.4 保险估值要点

1. **长期投资收益假设**：通常假设5%，过高（如5.5%+）会高估EV
2. **风险贴现率**：寿险通常10-11%，财险8-9%
3. **新业务结构**：高价值率产品（重疾险、长期储蓄）占比越高越好
4. **利率敏感性**：低利率环境压缩利差，对EV有负面影响

## 三、券商估值

### 3.1 券商业务特点

- **强周期性**：业绩与股市行情高度相关
- **轻资产vs重资产**：传统经纪轻资产，两融、自营重资产
- **杠杆弹性**：牛市加杠杆放大收益，熊市去杠杆

### 3.2 券商估值核心方法

#### 方法一：PB估值法（主推）

**核心逻辑**：与银行类似，但周期性更强

```
目标价 = BVPS × 目标PB

目标PB取决于：
- ROE水平和稳定性
- 业务结构（轻资产占比）
- 市场周期位置
```

**PB参考区间**：

| 市场环境 | PB区间 | 说明 |
|----------|--------|------|
| 熊市底部 | 0.8-1.2x | 布局时机 |
| 震荡市 | 1.2-1.8x | 正常水平 |
| 牛市 | 1.8-3.0x | 估值扩张 |
| 牛市顶峰 | 3.0x+ | 泡沫警惕 |

**业务结构溢价**：

| 业务特点 | PB调整 |
|----------|--------|
| 财富管理占比高 | +0.2-0.5x |
| 投行能力强 | +0.2-0.3x |
| 机构经纪占比高 | +0.1-0.2x |
| 自营占比高 | -0.1-0.3x（波动大） |

#### 方法二：PE估值法（辅助）

**注意周期调整**：
```
正常化PE = 市值 / 正常化净利润

正常化净利润 = 3-5年平均净利润，或
              = 中性市场假设下的净利润
```

**PE参考**：

| 市场环境 | PE区间 | 说明 |
|----------|--------|------|
| 熊市 | 10-15x | 盈利低谷，PE看似高 |
| 正常 | 15-25x | 合理区间 |
| 牛市 | 20-35x | 盈利高峰 |

### 3.3 券商估值关键指标

| 指标 | 定义 | 优秀 | 良好 | 警惕 |
|------|------|------|------|------|
| **ROE** | 净利润/净资产 | >12% | 8-12% | <6% |
| **经纪市占率** | 经纪收入/行业 | >5% | 2-5% | <1% |
| **投行市占率** | 承销金额/行业 | >5% | 2-5% | <1% |
| **财富管理收入占比** | 财富管理/总收入 | >20% | 10-20% | <10% |
| **杠杆率** | 总资产/净资产 | 3-4x | 4-5x | >5x |

### 3.4 券商估值要点

1. **周期判断最重要**：牛市高PB进入，熊市低PB布局
2. **业务结构转型**：从通道业务向财富管理、机构业务转型的券商应给予溢价
3. **头部集中**：政策支持头部券商，中小券商估值折价
4. **杠杆弹性**：牛市时高杠杆券商弹性更大，但风险也更高

## 四、金融行业估值检查清单

### 4.1 银行

- [ ] ROE与PB是否匹配
- [ ] 不良率是否真实（关注逾期90天+）
- [ ] 拨备是否充足
- [ ] NIM趋势如何
- [ ] 区域经济质量

### 4.2 保险

- [ ] EV计算假设是否合理（投资收益率、贴现率）
- [ ] 新业务价值率趋势
- [ ] 代理人数量和产能
- [ ] 退保率是否正常
- [ ] 投资资产配置（权益、非标占比）

### 4.3 券商

- [ ] 当前市场周期位置
- [ ] ROE的周期调整
- [ ] 业务结构（轻资产vs重资产）
- [ ] 市占率趋势
- [ ] 杠杆水平和风控能力
